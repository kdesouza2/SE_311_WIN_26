**Finite Automaton (FA)**
A Finite Automaton is a mathematical model of computation used to the represent systems with a finite number of states and deterministic or nondeterministic transitions between those states. 

A type of process control architecture

*Types of FA*
DFA (Deterministic FA)
    Exactly one transtion per input symbol from each state

NFA (Nondeterministic FA)
    Multiple possible transitions
